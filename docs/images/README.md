## Cách chụp ảnh

1. Chạy ứng dụng: `npm run dev`
2. Mở trình duyệt tại `http://localhost:3000`
3. Chụp ảnh màn hình theo từng bước trong guideline
4. Lưu vào thư mục này với đúng tên file quy định
5. Kích thước khuyến nghị: 1200x800px hoặc nhỏ hơn

## Ví dụ đường dẫn trong markdown

```markdown
![Mô tả](./images/01-settings-button.png)
```
